Parametric Approaches: Extreme Value

Created for Alchemists Academy by MacLane Wilkison

Introduction

Definition: A branch of applied statistics that focuses on the distinctiveness of extreme values

Generalized Extreme-Value Theory

  • Hξ,μ,σ=
    • exp[-(1+ξ[x-μ]/σ)-1/ξ], if ξ ≠ 0
    • exp[-exp([x-μ]/σ)], if ξ = 0
  • If ξ < 0, the GEV becomes a Frechet distribution
  • If ξ = 0, the GEV becomes a Gumbel distribution
  • If ξ < 0, the GEV becomes a Weibull distribution
'x' satisfies the condition where 1+ξ(x-μ)/σ > 0. 'μ' = the location parameter of the limiting distribution. 'σ' = the scale parameter of the limiting distribution. 'ξ' = the tail index (an indication of the shape of the tail of the limiting distribution.

Estimation of EV Parameters

  • Maximum likelihood (ML) methods
  • Regression methods
  • Semi-parametric methods

Peaks-Over-Threshold Approach: Generalized Pareto Distribution

  • Natural way to model exceedances over a high threshold
  • Fu(x) = Pr{X-u≤x|X>u} = [F(x+u)-F(u)]/[1-F(u)]
  • Gnedenko-Pickands-Balkema-deHaan (GPBdH):
    • If ξ≠0: Gξ,β(x) = 1-(1+ξx/β)-1/ξ
    • If ξ=0: Gξ,β(x) = 1-exp(-x/β)
As 'u' increases, the Fu(x) distribution converges to a generalized Pareto distribution given by GPBdH.

Refinements to EV Approaches

  • Conditional EV
  • Dealing with dependent (non-iid) data
  • Multivariate EVT
Conditional EV is applied when 'X' is adjusted for some dynamic structure. There are two simple solutions for dealing with non-iid data that exhibits clustering: (1) apply GEV distributions to block maxima and (2) estimate the tail of the conditional distribution rather than the unconditional one. Multivariate EVT can be used to model the tails of multivariate distributions in a theoretically appropriate way.

Limitations of EV

  • Relatively few observations leads to uncertain estimates
  • Considerable model risk
  • Sensitive to small sample effects, biases, non-linearities, and other problems
