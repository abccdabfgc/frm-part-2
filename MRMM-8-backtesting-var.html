Backtesting VaR

Introduction

  • Model validation - the general process of checking whether a model is adequate
  • Backtesting - a formal statistical framework that consists of verifying that actual losses are in line with projected losses
It is important to vet your VaR models for faulty assumptions, wrong parameters, or inaccurate modeling.

Setup for Backtesting

In a well-calibrated model, the number of observations falling outside VaR should be aligned with the confidence level

Model Backtesting with Exceptions

  • Classic statistical decision problem: accept or reject the hypothesis
  • Failure rate - the proportion of times VaR is exceeded in a given sample
  • Type I error - rejecting a correct model
  • Type II error - not rejecting an incorrect model
